# WiFiPassword

WiFiPassword 是一款 Mac 软件，有如下功能：

1. 忘记 mac 的 wifi 密码之后，可以查看 wifi 密码
2. mac 连接 WiFi 之后，可以生成二维码给手机扫描加入
3. 可以保存 WiFi 信息成图片，UI 还过得去


## 软件信息

1. 开发语言：Swift 5
1. UI：SwiftUI
1. 运行环境：Mac 10.15+
